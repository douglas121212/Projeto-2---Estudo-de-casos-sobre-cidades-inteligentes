# Projeto-2---Estudo-de-casos-sobre-cidades-inteligentes
Nas cidades inteligentes, a segurança pública integra IA e IoT, usando reconhecimento facial para monitorar indivíduos, prevenir crimes e agilizar respostas, promovendo um ambiente urbano mais seguro e eficiente, com análise de dados em tempo real e proteção da população.
